The Twins:

EVIDENCE: EMF5, Spirit Box, Thermometer

Hunt Threshold: 50%

Speed:1.5m/s or 1.9m/s (depends on which Twin hunts)

Hidden Ability: Both Ghosts can hunt. Main Ghost: 10% LESS speed Decoy: 10% MORE speed

Give Away: "Twinteraction"; Two interactions back to back in different rooms

Exclude: -

Ability :Twinteraction: refers to event where on the Van activity meter there are two different slope angles in one reading. Example Note if two interactions happen close to each other in time, but in multiple different rooms a single ghost could not reach so quickly.
